# Dual-Cluster-Contrastive

### Installation

```shell
cd Dual-Cluster-Contrast
python setup.py develop
```

### Prepare Datasets

```shell
cd examples && mkdir data
```
Download the person datasets Market-1501, and MSMT17 from [aliyun](https://virutalbuy-public.oss-cn-hangzhou.aliyuncs.com/share/data.zip), which is provided by existing public code ClusterContrast(https://github.com/alibaba/cluster-contrast).
Then unzip them under the directory like
```
examples/data/market1501/Market-1501-v15.09.15
exmpales/data/msmt17/MSMT17_V1
```
